買う前の確認

USB-C端子があっても、モニターに映るとは限りません

USB-Cハブやモバイルモニターを選ぶ前に、PC側が映像出力に対応しているかを確認する手順です。商品を探す前に3つだけ見ておくと、買い直しを減らせます。

運営・編集: はかどる机最終確認: 2026年7月19日編集方針

このページは対応条件を整理するための参考情報です。実際の動作はPC・ケーブル・ハブ・モニターの組み合わせで変わります。購入前に各メーカーの仕様を確認してください。

最初に確認する3つ

USB-Cハブの商品ページを見る前に、PC側の情報を用意します。

  1. 1
    PCの型番を控える

    本体底面のラベル、購入時の案内、Windowsの「設定」→「システム」→「バージョン情報」などで型番を確認します。型番が分かれば、メーカーの仕様表を探せます。

  2. 2
    USB-C端子の機能を仕様表で見る

    仕様表に「DisplayPort Alt Mode」「Thunderbolt」「USB4」「映像出力対応」などの表記があるか確認します。端子の形だけで判断しません。

  3. 3
    使いたい画面と充電条件を整理する

    モニターの解像度・リフレッシュレートと、PC付属充電器のW数を控えます。ハブ側は映像出力とPCへの最大給電の両方を確認します。

仕様表で探す言葉

見つからない場合は、型番とこの言葉を一緒に検索します。

映像出力

見る言葉
DisplayPort Alt Mode / 映像出力 / Thunderbolt / USB4
意味
USB-Cから外部画面へ映像を送れる可能性を確認する目安

画面の条件

見る言葉
4K / 60Hz / HDMI 2.0 など
意味
PC・ハブ・ケーブル・モニターのすべてで必要な条件を満たすか確認

給電

見る言葉
PD / Power Delivery / 最大給電W数
意味
ハブへ入る電力ではなく、PCへ送れる最大W数を確認

よくある勘違い

購入前に避けたい思い込みだけを短く整理します。

USB-C端子があれば、HDMI変換で必ず映りますか？

必ずしも映りません。PC側のUSB-C端子が映像出力に対応している必要があります。端子の形ではなく、型番ごとの仕様表を確認してください。

ハブに「4K対応」と書いてあれば、4K/60Hzで使えますか？

PC、ハブ、ケーブル、モニターがそれぞれ4K/60Hzなど必要な条件に対応している必要があります。どれか一つでも条件が足りないと、表示できる解像度や更新頻度が下がる場合があります。

100W対応のハブなら、PCを100Wで充電できますか？

充電器・ケーブル・ハブ・PCの対応範囲で決まります。商品ページの「最大100W入力」と「PCへの最大給電」を分けて確認します。
